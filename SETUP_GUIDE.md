# 📘 セットアップ詳細マニュアル

YouTubeアナリティクス分析ツールを、自分のチャンネルで使えるようにする手順です。**コピペメインで30〜45分**で完成します。

## 全体フロー

```
事前準備
   ↓
STEP 1: Googleスプレッドシート作成 + コード貼り付け（10分）
   ↓
STEP 2: APIサービスの有効化（3分）
   ↓
STEP 3: 初回実行＆認証（5分）
   ↓
STEP 4: ウェブアプリとしてデプロイ（3分）
   ↓
STEP 5: HTMLツール起動＆設定（5分）
   ↓
STEP 6: （任意）Claude API設定（15分）
   ↓
STEP 7: （任意）自動実行トリガー設定（2分）
   ↓
STEP 8: （任意）デスクトップアプリ化（5分）
   ↓
完成 🎉
```

## 事前準備

以下を確認：

- [ ] **Chromeブラウザ** がインストール済み
- [ ] **YouTubeチャンネルを運営しているGoogleアカウント** にログイン可能
- [ ] **クレジットカード**（STEP 6でClaude API使う場合のみ）

ブラウザで運営アカウントにログインしていることを確認してください。**別アカウントで作業すると後で詰みます。**

## STEP 1: スプレッドシート作成 + コード貼り付け

### 1-1. 新規スプレッドシート作成
1. https://docs.google.com/spreadsheets/ にアクセス
2. 「**+ 空白**」をクリック → 新規スプレッドシートが開く
3. 左上の「無題のスプレッドシート」をクリック → 名前を **`youtube_analytics`** などに変更

### 1-2. Apps Scriptエディタを開く
1. メニューバー「**拡張機能**」→「**Apps Script**」
2. 新しいタブで Apps Scriptエディタが開く

### 1-3. コードを貼り付け
1. エディタに表示されている「**コード.gs**」の中身を**全部削除**
2. このフォルダの `youtube_analytics_apps_script.gs` を**メモ帳**などで開く
3. 中身を全部コピー（Ctrl+A → Ctrl+C）
4. Apps Scriptエディタに貼り付け（Ctrl+V）

### 1-4. ⚠️ 重要：投稿トークンを変更

コードの上の方にこういう行があります：

```javascript
const REPLY_TOKEN = 'CHANGE_ME_TO_RANDOM_TOKEN_xyz123abc456';
```

この文字列を**自分で考えたランダムな文字列**（16文字以上推奨）に変更してください。

**例：**
- `youtube_secret_kj8z4xn7q`
- `mychannel_2026_token_abc`

⚠️ この文字列は後でHTMLツール側でも使うので、**メモしておく**こと。

### 1-5. 保存
- Ctrl+S または 💾 ボタンで保存

## STEP 2: APIサービスの有効化

スクリプトがYouTube APIを使えるようにします。

1. Apps Scriptエディタの**左側メニュー**を見る
2. 「**サービス**」という項目の横の「**+**」をクリック
3. 「**YouTube Data API v3**」を探して選択
4. 識別子は「**YouTube**」のまま → 「**追加**」
5. もう一度「**サービス**」の「**+**」をクリック
6. 「**YouTube Analytics API**」を探して選択
7. 識別子は「**YouTubeAnalytics**」のまま → 「**追加**」

### 確認

左メニューに以下が表示されていればOK：
- サービス
  - YouTube
  - YouTubeAnalytics

---

## STEP 3: 初回実行 ＆ 認証

スクリプトを初回実行して、Googleアカウントの権限を承認します。

### 3-1. 関数を選択して実行
1. エディタ上部の関数選択ドロップダウン（▶ボタンの右）から「**dailyUpdate**」を選択
2. 「**▶ 実行**」ボタンをクリック

### 3-2. 認証画面で許可
1. 「承認が必要です」というポップアップが出る
2. 「**権限を確認**」をクリック
3. **YouTubeを運営しているGoogleアカウント**を選択
4. 「**このアプリは確認されていません**」という警告が出る場合：
   - 「**詳細**」をクリック
   - 「**(プロジェクト名) に移動**」をクリック
5. 権限要求の一覧が表示される
6. 「**すべて選択**」にチェック
7. 「**続行**」をクリック

### 3-3. 再度実行
認証後は1回目の実行が止まっている場合があります。
- もう一度「**▶ 実行**」をクリック

### 3-4. 成功確認

スプレッドシートのタブに戻り、以下のシートが自動生成されているか確認：

- [ ] チャンネル概要
- [ ] 動画一覧
- [ ] 日次推移
- [ ] 視聴者層_年齢
- [ ] 視聴者層_性別
- [ ] トラフィックソース
- [ ] コメント
- [ ] 返信スタイル例
- [ ] 取得ログ

「取得ログ」シートの最新行に「**✅ 成功**」が記録されていればOK。

### よくあるエラーと対処

| エラー | 原因 | 対処 |
|---|---|---|
| `You do not have permission to call ...` | 認証で「すべて選択」を選ばなかった | Googleアカウント設定でアプリ削除→再実行 |
| `Unknown identifier (impressions)` | 既知の制限 | 無視してOK（CTRはCSVで補完） |
| 認証画面が出ない | 別アカウントでログイン中 | Googleアカウント切替 |

## STEP 4: ウェブアプリとしてデプロイ

HTMLツールがApps Scriptのデータを読み込めるよう、ウェブアプリとして公開します。

### 4-1. デプロイ作成
1. Apps Scriptエディタ右上の「**デプロイ**」→「**新しいデプロイ**」
2. 左上の歯車アイコン →「**ウェブアプリ**」を選択
3. 設定：
   - **説明**：「YouTube analytics」など（任意）
   - **実行ユーザー**：「**自分**」
   - **アクセス権**：「**全員**」
4. 「**デプロイ**」をクリック

### 4-2. URLをコピー

デプロイ完了画面で「**ウェブアプリ**」のURLが表示されます：

```
https://script.google.com/macros/s/AKfycb.../exec
```

このURLを**メモ**しておいてください。

### 「全員」って大丈夫？

「アクセス権=全員」と聞くと不安に思うかもしれませんが、以下の理由で安全です：

- URLは60文字以上のランダム文字列 → 推測不可能
- URLを公開しない限り、他人には絶対知られない
- 読み取り専用（YouTubeを操作したり、データを変更したりしない）
- 万が一漏れても、自分のチャンネルの数字を見られるだけ

## STEP 5: HTMLツールを起動して接続

### 5-1. HTMLを開く
1. このフォルダ内の `youtube_analytics_dashboard.html` を**Chromeにドラッグ&ドロップ**
2. または、Chromeのアドレスバーに以下を入力：
   ```
   file:///[ファイルのパス]/youtube_analytics_dashboard.html
   ```

### 5-2. データ取得
1. ツール上部の「Apps ScriptウェブアプリのURLを貼り付け」欄に、STEP 4でコピーしたURLを貼り付け
2. 「**📥 取得**」ボタンをクリック
3. 数秒後 → 「**✅ 取得成功**」と表示されればOK

### 5-3. チャンネル情報を入力
1. 「**⚙️ 設定**」タブを開く
2. 以下を入力：
   - **チャンネル名**：あなたのチャンネル名
   - **チャンネル概要**（1〜3行）：例「料理レシピを紹介するチャンネル」
   - **想定視聴者層**（任意）：例「30〜50代の女性」
3. **Apps Script投稿トークン**：STEP 1-4で決めた文字列を貼り付け
4. 「**💾 設定を保存**」

### 5-4. 動作確認
- 「**📊 ダッシュボード**」タブ：KPIが表示される
- 「**🎬 動画一覧**」タブ：全動画のリストが表示される
- 「**📈 パフォーマンス分析**」タブ：散布図が表示される
- 「**👥 視聴者層**」タブ：年齢・性別のグラフが表示される

ここまでで基本機能は完成です。**AI機能を使わない場合はここで終了**でも構いません。

## STEP 6: Claude API設定（任意・AI機能を使うため）

AI戦略アドバイザー、コメント返信生成などのAI機能を使う場合のみ必要です。

### 6-1. Anthropic アカウント作成
1. https://console.anthropic.com/ にアクセス
2. 「Sign up」でアカウント作成（Googleアカウントでも可）

### 6-2. APIキー発行
1. ログイン後、左メニュー「**API Keys**」
2. 「**Create Key**」をクリック
3. キー名を入力（例：`youtube-analytics`）→ 作成
4. **`sk-ant-api03-...`** で始まるキーが表示される
5. ⚠️ **「Copy」ボタンですぐコピー**（一度しか表示されない！）

### 6-3. クレジット購入
1. 左メニュー「**Plans & Billing**」
2. 「**Add credits**」をクリック
3. 「**Other**」を選択
4. **$5** と入力（最低額）
5. クレジットカード情報を入力 → 決済

月60〜400円程度の利用なので、$5で数ヶ月持ちます。

### 6-4. HTMLツールに設定
1. 「**⚙️ 設定**」タブ
2. **Claude APIキー**：先ほどコピーした `sk-ant-api03-...` を貼り付け
3. **モデル**：「Claude Sonnet 4.5（推奨）」のまま
4. 「**💾 設定を保存**」

### 6-5. 動作確認
1. 「**💡 自動インサイト＆改善提案**」タブ
2. 戦略提案③の「**🧠 包括的戦略レポートを生成**」ボタンをクリック
3. 30〜60秒待つ → 詳細な戦略レポートが表示されれば成功！

## STEP 7: 自動実行トリガー設定（推奨）

毎朝自動でデータ取得するように設定します。一度設定すれば以降不要。

1. Apps Scriptエディタに戻る
2. 左メニューの**時計アイコン**「**トリガー**」をクリック
3. 右下「**+ トリガーを追加**」
4. 設定：
   - 実行する関数：`dailyUpdate`
   - イベントのソース：**時間主導型**
   - 時間ベースのトリガーのタイプ：**日タイマー**
   - 時刻：**午前6時〜7時**
5. 「**保存**」

これで毎朝自動でデータ取得されます（PCがOFFでもOK）。

## STEP 8: デスクトップアプリ化（任意・おすすめ）

HTMLを「アプリっぽい独立ウィンドウ」で起動できるようにします。

### 8-1. HTMLを英字フォルダにコピー
日本語パスだとうまく動かないので、以下のような英字パスに置きます：

1. エクスプローラーで `C:` ドライブを開く
2. 新規フォルダ作成 →「**fmkk**」など英字名に
3. `youtube_analytics_dashboard.html` を `C:\fmkk\` にコピー

### 8-2. ショートカット作成
1. デスクトップで右クリック →「**新規作成**」→「**ショートカット**」
2. 項目の場所に以下を入力：

```
"C:\Program Files\Google\Chrome\Application\chrome.exe" --app="file:///C:/fmkk/youtube_analytics_dashboard.html"
```

3. 「**次へ**」 → 名前「**チャンネル分析**」など → 「**完了**」
4. デスクトップのアイコンをダブルクリック → 独立ウィンドウで起動

### 8-3. タスクバーにピン留め
1. 起動した状態でタスクバーのアイコンを右クリック
2. 「**タスクバーにピン留め**」

これで1クリックでツール起動できます。

## 🐛 トラブルシューティング

### HTMLで「Failed to fetch」エラー
**原因：**Apps Scriptのデプロイ設定が「自分のみ」になっている
**対処：**STEP 4を「**全員**」でやり直し → 新URLをHTMLに貼り直し

### Apps Scriptで「実行」ボタンがグレーアウト
**原因：**コードを保存していない
**対処：**Ctrl+S で保存

### コメント取得が失敗
**原因：**認証で必要な権限を許可していない
**対処：**Googleアカウント設定 → セキュリティ → サードパーティアプリ → 該当アプリのアクセス権を削除 → Apps Scriptで再実行

### CTR・インプレッションが0
**原因：**YouTube APIの仕様（公開APIで取得不可）
**対処：**YouTube StudioからCSVをエクスポート → HTMLにドラッグ&ドロップで補完

### 視聴者層グラフが空
**原因：**チャンネルが新しい・データ不足
**対処：**動画10本以上・公開90日以上経つと表示される

### AI機能でエラー「HTTP 401」
**原因：**Claude APIキーが間違っている、または期限切れ
**対処：**新しいキーを発行 → 設定に貼り直し

### AI機能でエラー「HTTP 429」
**原因：**Claude APIのレート制限
**対処：**1〜2分待ってから再試行
