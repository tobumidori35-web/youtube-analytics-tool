# 📊 YouTubeアナリティクス分析ツール

YouTube Studioのデータを自動取得して、**AIが具体的な改善案を提案する**個人YouTuber向け分析ツールです。

## ✨ 主な機能

### 自動取得（毎朝）
- 全動画の再生数・CTR・視聴維持率・収益
- 視聴者層（年齢・性別）
- トラフィックソース（どこから視聴者が来ているか）
- コメント一覧（未返信が分かる）

### AI機能（Claude API利用・月60〜400円程度）
- 🧠 **AI戦略アドバイザー**：全データを横断分析して、具体的な改善案・次に作るべき動画テーマを提案
- 💬 **戦略について深掘りチャット**：分析結果について追加質問が可能
- 🤖 **コメントAI返信生成**：あなたの過去返信スタイルを学習して、自然な返信案を作成
- 🎯 **動画別個別分析**：1動画ずつ「タイトル改善案・サムネコピー案」を出してくれる

### その他
- 📈 ダッシュボード（KPI一覧・日次推移・月別パフォーマンス）
- 🎬 動画一覧（CTR・維持率でフィルタ）
- 💡 自動インサイト（CTR詐欺動画の発見など）
- 💾 戦略レポートの保存・Markdownダウンロード

## ❓ よくある質問

**Q. プログラミング知識は必要？**
A. 不要です。コピペだけで設定できます。Claude Codeを使えば対話的に進められます。

**Q. 自分のチャンネルのデータが第三者に見られたりしない？**
A. すべてあなた個人のGoogleアカウント内で完結します。データは外部に送信されません。
（AI機能を使う時のみ、データの一部がAnthropicのClaude APIに送信されます）

**Q. データはどこに保存される？**
A. Googleスプレッドシート（あなた個人のGoogleドライブ内）に保存されます。
HTMLツール側の設定（APIキーなど）はブラウザのlocalStorageに保存されます。

**Q. 毎日決まった時間に自動でデータ取得させたい**
A. Apps Scriptに「トリガー」を1つ設定すれば、毎朝自動取得されます（PCを開いていなくてもOK）。
1. Apps Scriptエディタ左の**時計アイコン「トリガー」** →「**+ トリガーを追加**」
2. 関数 `dailyUpdate` ／ イベントのソース `時間主導型` ／ タイプ `日付ベースのタイマー` ／ 時刻 `午前6時〜7時` を選択 → **保存**
詳しい手順は SETUP_GUIDE.md の **STEP 7** を参照してください。

**Q. アンインストールしたい時は？**
A. 以下を削除すれば完全に消えます：
1. Googleドライブのスプレッドシート
2. このフォルダ
3. Google アカウント設定 → セキュリティ → 「サードパーティアプリ」から該当アプリのアクセス権を削除

**Q. 他のチャンネルにも導入できる？**
A. 一切問題ありません。同じ手順でセットアップすれば、何チャンネルでも分析できます。
